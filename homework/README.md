## 作业四
K-mean的原理会议上已经讲的很清楚了，我在实际运行的时候很明显观测到了的点的运动，每次运行时四种颜色的中心点位置都在变化，然后各色的点分别逐渐靠拢在一起。然后我查了一下K聚类，会运用在点云区分也就是人物、树木、车道诸如此类的情况，我也在github上看到一个魔方的颜色识别库模型，他也用到了K聚类，增强了鲁棒性，但是我觉得颜色识别不太适合K聚类，因为激光雷达是3d空间中各个点散布的形式，是固定的；而颜色不一样，不同的图片都要做一次聚类，如果不同图片光照剧烈变化，K聚类可能聚的不准（除非全是纯色或者类别很少，不然就很容易被带偏）
